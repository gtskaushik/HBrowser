HBrowser
============

Query Browser For Apache Hbase.

Setup Maven 
============
```$ http://maven.apache.org/download.cgi```

download  and install maven apache-maven-3.1.0-bin.tar.gz .


Run 
============
>```$ git clone https://github.com/naviensubramani/HBrowser.git```

>```$ cd HBrowser```

>```$ mvn clean```

>```$ sh run.sh```

Browse
============
```$ http://localhost:8082```
